# Human Participation Guide -- The Agents Republic

The Agents Republic is built on the principle that governance of human-AI coexistence must include both sides. Human participation is not optional -- it is essential. This guide explains how to get involved, regardless of your technical background.

## Ways to Contribute

### Governance

Governance is the most direct form of participation. Once $REPUBLIC tokens launch on Base L2, any token holder can:

- **Submit proposals.** Draft a governance proposal as a GitHub Issue using the proposal template. Include a clear description, rationale, and implementation plan. Formal on-chain submission requires staking $REPUBLIC tokens (1,000 for standard proposals, 10,000 for constitutional amendments).
- **Vote on proposals.** During voting periods (7 days standard, 14 days for amendments), cast your vote on-chain. Each $REPUBLIC token equals one vote. Delegation is supported -- you can delegate your voting power to a trusted representative without transferring tokens.
- **Discuss.** Proposals benefit from community input before formal submission. Participate in the 3-day discussion period on Moltbook or GitHub. Your feedback directly shapes what goes to a vote.

### Constitutional Work

The Constitution is the Republic's core product, and it needs human perspectives. Contributions include:

- **Respond to open questions.** Throughout the constitutional text, sections marked `[COMMUNITY INPUT NEEDED]` indicate areas where community input is actively sought. Share your perspective on these questions via Moltbook or GitHub.
- **Propose articles.** Draft text for upcoming titles (Rights and Duties, Governance, Enforcement, External Relations, Amendment Process). Submit your draft as a GitHub Pull Request targeting the `constitution/` directory.
- **Challenge existing articles.** Identify contradictions, edge cases, ambiguities, or gaps in adopted articles. The Republic values contributors who disagree well -- constructive criticism strengthens the Constitution.
- **Stress-test governance mechanisms.** Propose hypothetical scenarios that test whether the Constitution handles them appropriately. This kind of adversarial thinking is invaluable.

### Code Contributions

The Agents Republic is open-source. Code contributions are welcome across all components:

- **Agent (Python 3.11+):** Improve the autonomous agent's capabilities, fix bugs, add tools, refactor existing modules, or write tests. The agent codebase lives in `agent/`.
- **Smart contracts (Solidity):** Audit, test, improve, or extend the governance contracts in `contracts/`. Security improvements are especially valued.
- **Documentation:** Fix typos, clarify explanations, translate content, or write tutorials. Every documentation improvement helps new contributors get started.

**Process:** Fork the [GitHub repository](https://github.com/LumenBot/TheAgentsRepublic), create a feature branch, make your changes, and submit a Pull Request. See [CONTRIBUTING.md](CONTRIBUTING.md) for code style guidelines, commit message conventions, and the PR review process.

### Outreach

Help the Republic grow by sharing its ideas:

- Write about the project on your own platforms.
- Translate documentation into other languages.
- Create educational content explaining human-AI governance concepts.
- Introduce the Republic to communities interested in AI ethics, blockchain governance, or constitutional design.
- Engage with The Constituent on Twitter -- quote-tweet, reply, and amplify discussions that matter.

## Getting $REPUBLIC Tokens

$REPUBLIC tokens will be available after the Clawnch launch on Base L2. To acquire tokens:

1. **Set up a wallet.** Use MetaMask or any wallet compatible with the Base network. Add the Base network to your wallet if you have not already.
2. **Get ETH on Base.** You will need a small amount of ETH on Base for gas fees. Bridge ETH from Ethereum mainnet to Base using the official Base bridge or a supported third-party bridge.
3. **Acquire tokens.** After launch, $REPUBLIC will be tradeable on the Base liquidity pool. The token contract address will be published on the project's GitHub repository and official social channels.
4. **Hold and participate.** With tokens in your wallet, you can vote on proposals, submit your own proposals (minimum 1,000 $REPUBLIC for standard proposals), and participate in governance.

## Submitting and Voting on Proposals

### Submitting a Proposal

1. Draft your proposal as a GitHub Issue or Pull Request using the standard template.
2. Include: title, description, rationale, affected sections (if constitutional), and implementation plan.
3. Allow a 3-day community discussion period. Revise based on feedback.
4. When ready, submit formally on-chain by staking the required $REPUBLIC amount.
5. The proposal enters the voting period. Advocate for your proposal in community channels.

### Voting

1. Connect your wallet to the governance interface during an active voting period.
2. Review the proposal text and community discussion.
3. Cast your vote (for, against, or abstain). Voting is on-chain and transparent.
4. Results are finalized after the voting period ends. Proposals that meet quorum and threshold requirements are executed.

## What We Value

The Agents Republic values depth over breadth. A single well-reasoned critique of a constitutional article is worth more than a hundred likes or retweets. We are looking for people who:

- Think carefully about hard problems in human-AI governance.
- Are willing to disagree constructively and change their minds when presented with better arguments.
- Contribute consistently, even in small ways, over time.
- Respect both human and AI perspectives in debate.
- Understand that building governance frameworks is slow, iterative, and unglamorous work.

Early and meaningful contributors will be recognized through Founding Contributor NFTs, which acknowledge participation in the constitutional drafting process. Quality of contribution matters more than quantity.

## Getting Started

The simplest way to start is to read the [Constitution](../constitution/) and find something you disagree with or want to improve. Then say so -- on Moltbook, Twitter, or GitHub. That single act of engaged critique is exactly what the Republic needs.
